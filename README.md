# 🌿 Paradise Nursery - Shopping Cart Application

A React-based e-commerce web application for an online plant shop called Paradise Nursery. Users can browse plants, view details, add items to a shopping cart, and manage cart quantities.

## Features

- Browse plants grouped by categories (Aromatic, Medicinal, Succulents)
- Add/remove items from shopping cart
- Update item quantities
- Calculate total cart amount
- Responsive design

## Technologies Used

- React.js
- Redux Toolkit
- JavaScript (ES6+)
- CSS3

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
